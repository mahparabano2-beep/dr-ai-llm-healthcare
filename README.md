# AI Clinical Decision Support System (Dr. AI)

## Overview
This project explores an LLM-based clinical decision support workflow using patient symptom inputs. The notebook evaluates structured prompting approaches for diagnostic reasoning and response quality.

## Files
- `dr_ai_clinical_decision_support.ipynb` – main notebook
- `requirements.txt` – dependencies

## Tools Used
- Python
- Jupyter Notebook
- PMC-LLaMA
- PyTorch
- NLP methods
